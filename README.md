# Shipping Calculator Web Application

This project is a simple tool to calculate shipping costs and delivery estimates.  
It helps optimize supply chain management by providing customers with transparent shipping rates.

## Features
- Calculate shipping cost based on package weight and distance
- Provide estimated delivery time
- Extendable for multiple carriers and pricing models
